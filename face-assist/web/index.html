Documentation

System Requirements

  • Hardware: Raspberry Pi 5 (8GB) with m.2 HAT and Hailo8L chip
  • Operating System: Home Assistant OS

Model Requirements

  • Supported formats: .onnx, .tflite
  • Maximum model size: 500MB
  • Recommended: Models optimized for Hailo8L NPU

Image Requirements

  • Supported formats: JPG, PNG
  • Recommended resolution: 640x480 to 1920x1080
  • Face should be clearly visible and well-lit
  • Maximum image size: 10MB

Best Practices

  • Register multiple images per person for better accuracy
  • Use images with different angles and lighting conditions
  • Ensure face is unobstructed and clearly visible
  • Avoid heavily processed or filtered images
